# 🏦 Banking Management System

Application web de gestion bancaire développée avec **React (frontend)**, **Laravel (backend API)** et **MySQL (database)**.

---

## 🚀 Objectif

Ce projet permet de simuler un système bancaire simple incluant :

- gestion des clients
- gestion des comptes bancaires
- opérations financières (dépôt, retrait, virement)
- calcul des intérêts
- gestion des utilisateurs avec rôles

---

## 🛠️ Technologies

- Frontend : React.js
- Backend : Laravel (API REST)
- Database : MySQL
- Auth : Laravel Sanctum

---

## 👤 Rôles utilisateurs

- Admin : gestion complète du système
- Manager : gestion des clients et comptes
- Cashier : opérations de dépôt et retrait
- Auditor : consultation et audit

---

## ⚙️ Fonctionnalités

### 🔐 Authentification

- Login sécurisé
- Gestion des rôles
- Protection des routes

### 👥 Clients

- Création, modification, suppression
- Consultation des informations

### 💳 Comptes

- Ouverture de compte
- Consultation du solde
- Historique des comptes

### 💰 Transactions

- Dépôts
- Retraits
- Virements entre comptes

### 📊 Intérêts

- Calcul automatique des intérêts
- Mise à jour des soldes

---

## 🧱 Architecture

```text
React (Frontend)
      ↓
Laravel API (Backend)
      ↓
MySQL Database
```
